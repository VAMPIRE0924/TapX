## 状态规则

- `✅ 已完成`：代码已落地，并有当前代码审查、自动化测试或真实浏览器证据。
- `🟡 正在完成`：实现存在，但尚未对参考站逐项闭环，或仍缺真实交互/数据验证。
- `⏸ 后端阶段`：前端形态已明确，缺口属于控制面 API、DB 或运行时执行。
- 只有一条具体能力可以标记完成；不得用“字段存在”推导“整页完成”。
- 监听端和连接端必须按“运行模式 -> 协议 -> 每个开关打开后的条件字段”逐项验收。
